Seniors Outdoors Redesign

Information architecture, Axure, user testing, HTML/CSS, Wordpress/PHP, JavaScript/jQuery

The Project

As a volunteer project, I redesigned the website for Seniors Outdoors, a club for active seniors in Durango, Colorado.

The Problem

The club's current site is dated and has a shallow structure with long pages of dense text, which makes content on the site difficult to access. In addition, club members are increasingly expressing a desire to be able to access information from their mobile devices, which the current site does not support.

The Approach

The project included a number of activities:

  1. Content inventory and proposed restructuring of the site's information architecture
  2. User testing consisting of a card sort and tree test to validate and refine the proposed IA
  3. Development of both paper & pencil and Axure prototypes
  4. Development of the redesigned site using WordPress (in progress)

The Result

The redesigned site is currently being developed using WordPress. Planned features of the redesigned site are:

  • Use of a calendar format for the club's outing schedule
  • Improved implementation of the photo gallery
  • Secure treatment of member information
  • A user-friendly interface to allow club organizers to add and modify content
  • Improved performance for mobile users
