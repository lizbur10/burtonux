Seniors Outdoors Redesign

Information architecture, Axure, user testing, HTML/CSS, Wordpress/PHP, Javascript/JQuery

The Project

As a volunteer project, I redesigned the website for Seniors Outdoors, a club for active seniors in Durango, Colorado.

My Role

Solo project

Website Redesign

Goals for the redesign:

  • Make the most important information on the site easier to find and access, specifically:
    • the outing schedule,
    • the image gallery, and
    • the member directory.
  • Improve the functionality of the site for smart phone and tablet users.

Activities

  • Performed a content inventory of the current site
  • Created a proposed site map
  • Conducted a card sort and tree testing to validate site organization
  • Developed prototypes of the proposed design

Methods & Tools

  • Content inventory
  • Creation of site map using PowerPoint
  • OptimalWorkshop tools OptimalSort and Treejack
  • Recruitment of participants via Amazon's Mechanical Turk
  • Prototype development using paper & pencil and Axure

Website development

Development of the revised site is currently in progress.

Development details

WordPress will be used to host the site in order to:

  • allow a calendar format to be used for the club's outing schedule,
  • provide for secure treatment of member information,
  • improve the implementation of the image gallery on the site, and
  • provide a user-friendly interface for club organizers to add and modify content.

Methods

  • Custom WordPress theme development.
  • Using PHP to customize plugins for the calendar, image gallery, and membership directory.
  • PHP and Javascript/JQuery programming of home page image slider.
