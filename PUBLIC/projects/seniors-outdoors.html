Seniors Outdoors Redesign

Information architecture, prototyping (paper, Axure), user testing (card sort, reverse card sort)

The Project

As a freelance project, I redesigned the website for Seniors Outdoors, a club for active seniors in Durango, Colorado.

The Problem

The club's previous site was dated and had a shallow structure with long pages of dense text, making content on the site difficult to access. In addition, club members were increasingly expressing a desire to be able to access information from their mobile devices, which the previous site did not support.

The Approach

The project included a number of activities:

  1. Content inventory and proposed restructuring of the site's information architecture
  2. User testing consisting of a card sort and tree test to validate and refine the proposed IA
  3. Development of both paper & pencil and Axure prototypes
  4. Development of the redesigned site using WordPress

The Result

The redesigned site was developed using WordPress. Features of the redesigned site include:

  • A fresher, more modern look
  • Dynamic presentation of upcoming outings
  • A calendar view for the club's outing schedule
  • A mechanism for uploading outings from a spreadsheet
  • A user-friendly interface to allow club organizers to add and modify content
  • Improved accessibility and performance for mobile users

My Role

Solo project: I did all research, design and development work.
