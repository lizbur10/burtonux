Class Project

Knitty.com

Information architecture, HTML/CSS, responsive design

The Project

I used the Knitty.com website as a project for both the Information Architecture class at Bentley University, and the Responsive Design class at the Minneapolis College of Art and Design.

My Role

Solo project

Information Architecture

Goals for the restructuring:

  • Surface the freshest content on the site.
  • Simplify the navigation and standardize it across all areas of the site.

Activities

  • Performed a content inventory and expert review of the current site
  • Created a proposed site map
  • Created interactive prototypes of the proposed redesign

Methods & Tools

  • Content inventory
  • Expert review
  • Creation of site map using PowerPoint
  • Creation of interactive prototype using paper & pencil and Axure

Mobile-first redesign & development

Goals for the project:

  • Create a mobile-first redesign of the landing page
  • Develop a coded prototype of the redesign

Design Details

  • Increased emphasis on content
  • Simplified navigation
  • Adequately-sized interactive elements
  • Options for user interactions with site content (e.g., favorite, share, comment)

Methods & Tools

  • HTML/CSS
  • JavaScript/JQuery
  • SASS
  • Fluid layout
  • Media queries
